# IT SYstem Logger

It is react based web application ,we can manage the people's tasks and data will come from json server.

## Local Setup

```sh
$ git clone git@github.com:adhit7/it-system-logger.git
```

```sh
$ cd it-system-logger

```

```sh
$ npm install
```

## Run Locally

```sh
$ npm start
```
## Run Json Server

```sh
$ npm run json-server
```
